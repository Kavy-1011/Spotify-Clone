🎵 Spotify Clone (Frontend)

A frontend-only Spotify Clone built using HTML and CSS. This project recreates the look and feel of Spotify's web player for learning UI development and responsive web design.

 ✨ Features

- 🎨 Spotify-inspired UI
- 📱 Responsive Layout
- 🎵 Home Screen Design
- 📚 Library Section
- 🔥 Trending Songs Cards
- 🎼 Recently Played Section
- ▶️ Static Music Player UI

🛠️ Tech Stack

- HTML5
- CSS3

 Purpose

The goal of this project was to practice:

- HTML Structure
- CSS Flexbox & Grid
- Responsive Web Design
